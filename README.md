# Rating
Получение топ‑5 фильмов из рейтинга «250 лучших фильмов».

## Функционал

- отправляет запрос на страницу топ‑250 Кинопоиска;
- парсит данные о первых 5 фильмах;
- выводит информацию в читаемом формате;
- обрабатывает возможные ошибки (сетевые, парсинга).

Данные:
- позиция в топе (№);
- название фильма;
- год выпуска;
- рейтинг;
- жанр.

## Зависимости

Для работы требуются следующие пакеты Python:
- `requests`
- `beautifulsoup4`
- `pandas`
